
# Setup

This is what I like to install. Stored here for easier access between machines.

### Brew

Install homebrew

```console
curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh > install-brew.sh
./install-brew.sh
```

Install all brew packages

```
brew bundle
```
